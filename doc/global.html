parseValue(value) → {*}

Source:
  • index.js, line 18
Parses input value to JS data type or structure, if it is possible. If you wrap the string in extra quotation marks, the value will not be parsed, and additional quotation marks will be deleted.
Example
parseValue('"[1, 2, 3]"');
  returns '[1, 2, 3]';

parseValue('["0.5", -Infinity, null, 0o377, { name: "alex", age: 22 }]');
  returns ['0.5', -Infinity, null, 255, { name: 'alex', age: 22 }];
Parameters:

value * it's a string usually
Returns:
- a JS data type or structure if the parsing was successful, otherwise the input value.

parseArg(arg, prefixesopt) → {parseResult}

Parses a string into a prefix, key, value
Example
parseArg('--foo=25', ['-']); // prefixes = ['-'] by default.
  returns { prefix: '--', key: 'foo', value: '25', arg: '--foo=25' };

parseArgs(argsopt, callbackopt, prefixesopt) → {Object.<string, *>}

Source:
  • index.js, line 89
Creates an object from the results of the callback call. Works like Array.prototype.map.
Example
parseArgs(['--foo=25', '-bar', '_qux='], (key, value) => {
  if (key === 'foo') return { key: 'f', value: value + 5 };
  return { key, value };
}); // prefixes = ['-'] by default.

returns { f: 30, bar: undefined, '_qux': '' };
Parameters:
Name Type Attributes Default Description
args Array.<string> <optional>
 process.argv array of arguments.
callback parseCallback <optional>
 (key, value) => ({ key, value }) a function that will be called for each of the array arguments with the following arguments: (key, value, prefix, arg), and the results of its calls will be written to the resulting object. If function returns object which contains key and value properties then result of its call will be written to the resulting object else result will not be written.
prefixes Array.<string> <optional>
 ['-'] array of strings which can be prefixes.
Returns:
result object with the following structure: { <key>: <value> }.

Object.<string, *>

argsParser(argsopt, optionsopt, keysopt) → {Object.<string, *>}

Source:
  • index.js, line 148
Parses each argument from array of arguments.
Example
const args = ['--foo=25', '__bar', '-Q'];
const options = { defaultValue: 500, prefix: '_' };
const keys = {
  qux: { aliases: ['Q'], defaultValue: true, prefix: '-' }
};

argsParser(args, options, keys);
  returns { bar: 500, qux: true };
Parameters:
Name Type Attributes Default Description
args Array.<string> <optional>
 process.argv array of arguments.
options Options <optional>
 Options the options object, sets global options for all arguments (keys).
keys Object.<string, KeyDescription> <optional>
 {} the dictionary object with the following structure: { <key>: <keyDescriptor> }.
Returns:
- result object with the following structure: { <key>: <value> }.
